Это учебный проект, демонстрирующий реализацию паттерна Model-View-Controller (MVC) на Python. Система предназначена для управления коллекцией рецептов с поддержкой различных ролей пользователей, фильтрации, поиска и статистики.

Проект создан в рамках домашнего задания по изучению архитектурных паттернов и принципов SOLID.

### Создайте виртуальное окружение (рекомендуется):
##### Windows
```
python -m venv venv
venv\Scripts\activate
```
##### Linux/Mac
```
python3 -m venv venv
source venv/bin/activate
```

### Запустите приложение:
```
python main_recipe.py
```
